# Changelog

## 0.0.1 — Initial public alpha

- Added Base, Extended and Full CSS bundles.
- Added an optional dependency-free JavaScript bundle for interactive components.
- Added semantic light, dark and system themes controlled by CSS custom properties.
- Added responsive layout, grid, forms, buttons, tables and utilities.
- Added cards, alerts, badges, navigation, overlays, disclosure components and app layout.
- Added mobile and iOS compatibility improvements for viewport sizing, safe areas and scroll locking.
- Added GitHub Actions CI, GitHub Pages deployment and release workflows.
- Added public API, accessibility, browser support, customization and contribution documentation.

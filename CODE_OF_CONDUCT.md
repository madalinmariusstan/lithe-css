# Code of Conduct

## Our standard

Be respectful, constructive and welcoming. Focus criticism on code and ideas, not people. Harassment, discrimination, threats and personal attacks are not tolerated.

## Scope

This standard applies to issues, pull requests, discussions and other project spaces.

## Enforcement

Project maintainers may edit, remove or reject contributions and participation that violate this standard. Serious concerns should be reported privately to the maintainer.

# Getting started

Lithe CSS is class-based and non-invasive. Native elements are not styled as components unless an explicit Lithe class is present.

## Full bundle

```html
<link rel="stylesheet" href="dist/lithe.min.css">
<script src="dist/lithe.js" defer></script>
```

## Modular bundle

```html
<link rel="stylesheet" href="dist/lithe-base.min.css">
<link rel="stylesheet" href="dist/lithe-extended.min.css">
<script src="dist/lithe.js" defer></script>
```

Use Base for layout, forms, buttons, tables and utilities. Add Extended for cards, feedback, navigation, overlays, disclosure and app layouts.

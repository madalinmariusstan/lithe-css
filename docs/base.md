# Base

Base contains design tokens, the minimal reset, layout primitives, grid, forms, buttons, tables and utilities.

It deliberately avoids global component selectors such as `button {}`, `input {}`, `table {}` and `article {}`.

```html
<div class="container stack">
  <div class="row">
    <div class="col-12 col-md-6">First</div>
    <div class="col-12 col-md-6">Second</div>
  </div>
</div>
```

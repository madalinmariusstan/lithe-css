# Release checklist

- [ ] Build Base, Extended, and Full bundles.
- [ ] Run automated checks.
- [ ] Validate JavaScript syntax.
- [ ] Test light and dark themes.
- [ ] Test 320 px, 375 px, 768 px, 1024 px, and wide desktop layouts.
- [ ] Test Chrome, Edge, Firefox, and Safari.
- [ ] Test iOS Safari with Reduce Motion both enabled and disabled.
- [ ] Test keyboard-only navigation.
- [ ] Test VoiceOver or another screen reader.
- [ ] Verify package version, changelog, and npm file list.

## Browser matrix before 1.0

- [ ] Chrome desktop and Android
- [ ] Edge desktop
- [ ] Firefox desktop
- [ ] Safari macOS
- [ ] Safari iOS on a physical device
- [ ] VoiceOver focus order for dialog, drawer, dropdown and tabs
- [ ] Reduce Motion spinner and panel behavior
- [ ] Safe-area layout in portrait and landscape
